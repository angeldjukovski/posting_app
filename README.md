                          Posting_App

A full_stack application with the idea of publushing and interaction between posts and users secure with user authentication.

The project is divided into two parts 

poster-front:
Poster front contains the web application of the project and the browser features. 
It was mainly done on NextJS Tailswind as framework other dependecies being used such as : Axios for API resquests and token intreceptor, and JWT for authentification.

The content of poster-front is distributed into folders such as components,context,hooks,lib,services types ect

poster-back :
Poster back constains the backend of the project such as the Database it is mainly developed on the nestJS framework and build on the principle of MVC(Model View Control) which means most of the content it is usally seperated into services, controllers, and moudles.
Other dependecies are being used like JWT bcrypt and docker plus typeORM, Database of the project is done on postgreSQL
